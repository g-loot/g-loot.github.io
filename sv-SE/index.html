Get started

How does it work?

Start G-Loot SDK

User is suggested to play on G-Loot via a call-to-action.
This will start our webapp inside the game.

Play match for money

Our webapp suggests to play a game for money.
This will create a match on our server, allowing players to compete against each others

Start Match In Game

A player is instantly invited to play, there is no lobby or queue.
At the end of the game, a report will be send to our servers

A winner is chosen

At the end of the match, our system will deduce if enough players joined in a given time frame to end the match.
Then, it will finalize the scores and stakes.
Winner(s) receive money in their wallet.
